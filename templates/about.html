About

This model is developed used for detecting the diseases which can occur on tomato plants using the tomato leaf images.

Details about the model

  • Technology Used: Convolutional Neural Networks
  • Accuracy: Correct prediction given 97 times out of 100
  • Dataset Used: Labelled Tomato Leaf Images
  • Dataset Source: Kaggle
  • Programming Language: Python
  • Model: ResNet152V2

The problem specified is multiclass image classifaction. The model can predict the disease of the tomato plant when given a tomato leaf image as input.

This prediction model can be used by farmers, agricultural scientists, biologists, gardeners and by the general audience to check the disease of the plant. This prediction can be useful to save the plant by giving the right treatment for the predicted disease.
